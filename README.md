# Дипломная работа: Определение уязвимых групп населения и факторов бедности в регионах России

Данный репозиторий содержит полный код и анализ дипломного проекта, выполненного в марте 2026 года на основе открытых данных Росстата, Генпрокуратуры и других ведомств.

## Описание проекта

Проект выполнен по заказу некоммерческой организации (НКО), планирующей открывать филиалы в регионах России для помощи малообеспеченным слоям населения.

**Цель работы:** Провести анализ социально-экономического положения регионов России для выявления территорий и групп населения, наиболее остро нуждающихся в поддержке, а также определить ключевые факторы, связанные с уровнем бедности.

## Задачи исследования

1.  **Кластеризовать регионы России** по уровню социального неблагополучия.
2.  **Описать группы населения, сталкивающиеся с бедностью** (дети, пенсионеры, трудоспособные).
3.  **Определить факторы**, влияющие на уровень бедности (экономические, демографические, социальные).

## Основные выводы и результаты

### 1. Кластеризация регионов
Выделено **4 устойчивых типа регионов**:
*   **Кластер 0 (Среднестатистические)** — 56% регионов.
*   **Кластер 1 (Регионы-лидеры)** — 10% регионов (Москва, СПб, Татарстан и др.).
*   **Кластер 2 (Проблемные с низкими доходами)** — **18% регионов** (территории с максимальной бедностью, низкой занятостью, нуждающиеся в первоочередной помощи).
*   **Кластер 3 (Северные и дальневосточные)** — 16% регионов.

### 2. Структура малоимущего населения
*   **Дети** — **37%**
*   **Трудоспособное население** — **53%**
*   **Пенсионеры** — **5%**

**Вывод:** Бедность в России — это проблема семей с детьми и работающего населения, а не пенсионеров.

### 3. Ключевые факторы бедности
*   Наиболее сильная обратная связь зафиксирована с **розничным оборотом** (r = -0.76), **доходами** (r = -0.55) и **занятостью** (r = -0.58).
*   Положительная корреляция с долей социальных расходов (r = 0.51) — это реакция бюджетов на проблемы, а не их причина.
*   Связь с преступностью и младенческой смертностью оказалась слабой, что может объясняться особенностями региональной статистики.
